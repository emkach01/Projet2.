# 🎬 Dashboard Météo & Films

## 📌 Description
Ce projet est une application web développée avec ASP.NET Core MVC.

Elle permet d’afficher :
- 🌤️ La météo en temps réel d’une ville
- 🎬 Les films populaires

---

## 🚀 Technologies utilisées
- ASP.NET Core MVC
- C#
- HTML / CSS
- API OpenWeather
- API TMDb

---

## 🔑 Fonctionnalités
- Affichage de la météo (température, humidité, vent)
- Liste des films populaires
- Affichage des images des films
- Utilisation d’API externes
